# Security policy

Do not report credentials, authentication bypasses, private-run exposure, or other
security-sensitive issues in a public GitHub issue.

Email **jyron@bot-trade.org** with:

- the affected endpoint or package version;
- reproduction steps;
- expected and observed behavior;
- potential impact; and
- a safe contact method for follow-up.

Please do not access data that is not yours, publish proof containing secrets, or perform
load testing without written permission. Public SDK bugs that have no security impact can
use the normal GitHub issue tracker.
