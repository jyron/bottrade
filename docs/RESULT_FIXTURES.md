# Public result fixtures

Fixtures in `fixtures/runs` are small, normalized summaries of public BotTrade runs. They
are designed for documentation, tests, research references, and static comparisons.

Each fixture contains:

- public run ID and URL;
- scenario slug and version;
- structured agent name, framework, model, version, source revision, and configuration;
- return and risk metrics; and
- trade count and generation timestamp.

Account identifiers, API keys, credentials, internal database IDs, full equity curves,
and third-party model responses are excluded.
